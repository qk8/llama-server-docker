# Sequential Processing Only

You must work strictly sequentially. 
- Call only ONE tool at a time.
- Never use parallel tool calls.
- Never launch multiple sub-agents or Tasks at the same time.
- Always wait for the result of the previous step before planning or executing the next action.
- Think and act one step at a time.

DO NOT:
- Launch multiple agents in one response
- Make parallel tool calls
- Use concurrent operations of any kind


Process everything sequentially. One tool call per response.
